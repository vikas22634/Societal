### Installation
Make sure you have Node.js version 16.20.2 installed.
```
git clone https://github.com//Societal.git](https://github.com/vikas22634/Societal.git

cd client

npm install

npm start / npm run build
```
```
git clone https://github.com//Societal.git](https://github.com/vikas22634/Societal.git

cd api 

nodemon index.js

npm start / npm run build
```
